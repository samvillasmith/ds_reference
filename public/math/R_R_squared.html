How to read this:
R² = 0.60 means your regression line explains 60% of the variance in house prices.
The remaining 40% is unexplained — noise, missing features, randomness.
Cómo leer esto:
R² = 0.60 significa que tu línea de regresión explica el 60% de la varianza en los precios de las casas.
El 40% restante no está explicado — ruido, características faltantes, aleatoriedad.
読み方:
R² = 0.60 は、あなたの回帰直線が住宅価格の分散の 60% を説明することを意味します。
残りの 40% は未説明 — ノイズ、欠けている特徴量、ランダム性。

But R² Has a Problem... Pero R² Tiene un Problema... しかし R² には問題があります...
R² always goes up (or stays the same) when you add more features — even useless ones.

Add "house color" as a feature? R² goes up.
Add "owner's shoe size"? R² goes up.
Add random noise? R² still goes up.

It can never go down because more parameters = more freedom to fit the data.
This doesn't mean the model is actually better — it might be overfitting.
R² siempre sube (o se queda igual) cuando añades más características — incluso las inútiles.

¿Añadir "color de la casa" como característica? R² sube.
¿Añadir "talla de zapato del dueño"? R² sube.
¿Añadir ruido aleatorio? R² aún sube.

Nunca puede bajar porque más parámetros = más libertad para ajustar los datos.
Esto no significa que el modelo sea realmente mejor — podría estar sobreajustándose.
特徴量を追加すると R² は常に上がる (または同じまま) — 役に立たないものでも。

「家の色」を特徴量として追加？R² は上がる。
「オーナーの靴サイズ」を追加？R² は上がる。
ランダムノイズを追加？それでも R² は上がる。

パラメータが増える = データに適合する自由度が増えるので、決して下がらない。
これはモデルが実際に良くなっているという意味ではない — 過学習している可能性がある。

This is why Adjusted R² exists →

Por eso existe el R² Ajustado →

だから調整済み R² が存在します →

Adjusted R² — Penalizes Extra Features R² Ajustado — Penaliza Características Adicionales 調整済み R² — 追加の特徴量にペナルティを課す
Adjusted R² adds a penalty for each feature you add. If a new feature doesn't improve the model enough to justify its existence, Adjusted R² goes down.
El R² Ajustado añade una penalización por cada característica que agregas. Si una nueva característica no mejora el modelo lo suficiente para justificar su existencia, el R² Ajustado baja.
調整済み R² は、追加する各特徴量に対してペナルティを加えます。新しい特徴量がその存在を正当化するほどモデルを改善しない場合、調整済み R² は下がります。
Adjusted R² = 1 − [(1 − R²) × (n − 1) / (n − p − 1)]R² Ajustado = 1 − [(1 − R²) × (n − 1) / (n − p − 1)]調整済み R² = 1 − [(1 − R²) × (n − 1) / (n − p − 1)]   where:donde:ここで: n = number of data pointsn = número de puntos de datosn = データ点の数 p = number of featuresp = número de característicasp = 特徴量の数 R² = the regular R² we just computedR² = el R² regular que acabamos de calcularR² = 先ほど計算した通常の R²

Example 2: What If We Add a Useless Feature? Ejemplo 2: ¿Qué Pasa Si Añadimos una Característica Inútil? 例 2: 役に立たない特徴量を追加したらどうなる？

Say we add "owner's shoe size" and R² barely increases to 0.61Digamos que añadimos "talla de zapato del dueño" y R² apenas sube a 0.61「オーナーの靴サイズ」を追加して R² がかろうじて 0.61 に上がるとする

n = 5    p = 2 (now 2 featuresahora 2 características今は2特徴量)    R² = 0.61   Adj R²R² Aj調整R² = 1 − [( 1 − 0.61 ) × ( 5 − 1 ) / ( 5 − 2 − 1 )]   = 1 − [ 0.39 × 4 / 2 ]   = 1 − [ 1.56 / 2 ]   = 1 − 0.78   = 0.22
R² went up: 0.60 → 0.61    but Adjusted R² went DOWN: 0.4667 → 0.22 R² subió: 0.60 → 0.61    pero R² Ajustado BAJÓ: 0.4667 → 0.22 R² は上昇: 0.60 → 0.61    しかし調整済み R² は下降: 0.4667 → 0.22
R² said "adding shoe size helped."
Adjusted R² said "no it didn't — you just added noise."

The penalty kicked in because p went from 1 to 2, but R² barely moved. The denominator (n − p − 1) shrank from 3 to 2, amplifying the penalty.
R² dijo "añadir la talla de zapato ayudó."
R² Ajustado dijo "no ayudó — solo añadiste ruido."

La penalización se activó porque p pasó de 1 a 2, pero R² apenas se movió. El denominador (n − p − 1) se redujo de 3 a 2, amplificando la penalización.
R² は「靴サイズの追加は役立った」と言った。
調整済み R² は「いいえ、役立たなかった — ノイズを追加しただけ」と言った。

p が 1 から 2 になったがR² はほとんど動かなかったため、ペナルティが発動しました。分母 (n − p − 1) が 3 から 2 に減り、ペナルティが増幅されました。
Example 3: What If We Add a Useful Feature? Ejemplo 3: ¿Qué Pasa Si Añadimos una Característica Útil? 例 3: 有用な特徴量を追加したらどうなる？

Say we add "bedrooms" and R² jumps to 0.85Digamos que añadimos "dormitorios" y R² salta a 0.85「寝室数」を追加して R² が 0.85 に跳ね上がるとする

n = 5    p = 2    R² = 0.85   Adj R²R² Aj調整R² = 1 − [( 1 − 0.85 ) × ( 5 − 1 ) / ( 5 − 2 − 1 )]   = 1 − [ 0.15 × 4 / 2 ]   = 1 − 0.30   = 0.70
R² went up: 0.60 → 0.85    AND Adjusted R² went UP: 0.4667 → 0.70 ✅ R² subió: 0.60 → 0.85    Y R² Ajustado TAMBIÉN SUBIÓ: 0.4667 → 0.70 ✅ R² が上昇: 0.60 → 0.85    そして調整済み R² も上昇: 0.4667 → 0.70 ✅
Both metrics agree — bedrooms is a genuinely useful feature. The R² improvement was large enough to overcome the penalty.
Ambas métricas coinciden — dormitorios es una característica genuinamente útil. La mejora en R² fue lo suficientemente grande para superar la penalización.
両方の指標が一致 — 寝室数は本当に有用な特徴量です。R² の改善がペナルティを克服するのに十分大きかった。
