1 · Descriptive Statistics
Mean (average)
μ = (1/n) × Σxᵢ
Sum all values, divide by count.
Median
Middle value when sorted
If n is even: average of the two middle values.
Mode
Most frequently occurring value
Can have 0, 1, or multiple modes.
Range
Range = max − min
Variance (population)
σ² = (1/n) × Σ(xᵢ − μ)²
Average squared deviation from the mean.
Variance (sample)
s² = (1/(n−1)) × Σ(xᵢ − x̄)²
Bessel's correction: divide by n−1 for unbiased estimate.
Standard Deviation
σ = √σ²    (population)
s = √s²    (sample)
Same units as the original data.
Standard Error of the Mean
SE = s / √n
How much the sample mean varies from sample to sample.
Covariance
Cov(X,Y) = (1/(n−1)) × Σ(xᵢ − x̄)(yᵢ − ȳ)
Positive = move together. Negative = move opposite. Scale-dependent.
Pearson Correlation
r = Cov(X,Y) / (sₓ × sᵧ)
Normalized covariance. Range: [−1, +1]. Scale-independent.
IQR (Interquartile Range)
IQR = Q3 − Q1
Outlier bounds: [Q1 − 1.5·IQR, Q3 + 1.5·IQR]
Z-Score (standardization)
z = (x − μ) / σ
How many std deviations x is from the mean.
2 · Probability
Basic Probability
P(A) = favorable outcomes / total outcomes
Always between 0 and 1.
Complement
P(A') = 1 − P(A)
Probability of A NOT happening.
Addition Rule
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
If mutually exclusive: P(A ∪ B) = P(A) + P(B)
Multiplication Rule
P(A ∩ B) = P(A) × P(B|A)
If independent: P(A ∩ B) = P(A) × P(B)
Conditional Probability
P(A|B) = P(A ∩ B) / P(B)
Probability of A given B has occurred.
Bayes' Theorem
P(A|B) = P(B|A) × P(A) / P(B)
Update beliefs with evidence. Foundation of Naive Bayes.
Law of Total Probability
P(B) = Σ P(B|Aᵢ) × P(Aᵢ)
Sum over all partitions of the sample space.
Expected Value
E[X] = Σ xᵢ × P(xᵢ)
Weighted average of all possible outcomes.
3 · Key Distributions
Normal (Gaussian)
f(x) = (1 / (σ√(2π))) × e^(−(x−μ)² / (2σ²))
Bell curve. 68% within ±1σ, 95% within ±2σ, 99.7% within ±3σ.
Bernoulli
P(X=1) = p    P(X=0) = 1−p
Single coin flip. Mean = p, Var = p(1−p).
Binomial
P(X=k) = C(n,k) × p^k × (1−p)^(n−k)
k successes in n independent trials. Mean = np, Var = np(1−p).
Poisson
P(X=k) = (λ^k × e^(−λ)) / k!
Count of events in fixed interval. Mean = Var = λ.
Uniform (continuous)
f(x) = 1 / (b − a)    for a ≤ x ≤ b
Equal probability everywhere. Mean = (a+b)/2.
Exponential
f(x) = λ × e^(−λx)    for x ≥ 0
Time between Poisson events. Mean = 1/λ.
4 · Hypothesis Testing
Z-Test Statistic
z = (x̄ − μ₀) / (σ / √n)
When population σ is known (or n ≥ 30).
T-Test Statistic
t = (x̄ − μ₀) / (s / √n)
When population σ is unknown. Uses t-distribution with n−1 df.
Two-Sample T-Test
t = (x̄₁ − x̄₂) / √(s₁²/n₁ + s₂²/n₂)
Compare means of two independent groups.
Chi-Square Statistic
χ² = Σ (Oᵢ − Eᵢ)² / Eᵢ
O = observed, E = expected. Tests categorical associations.
Confidence Interval
x̄ ± z* × (s / √n)
95% CI: z* = 1.96. 99% CI: z* = 2.576.
p-value
P(data ≥ observed | H₀ is true)
If p < α (usually 0.05), reject H₀.
Type I error (α): rejecting H₀ when it's actually true (false positive).
Type II error (β): failing to reject H₀ when it's actually false (false negative).
Power = 1 − β: probability of correctly rejecting a false H₀.
5 · Linear Regression
Simple Linear Regression
ŷ = θ₀ + θ₁x
θ₀ = intercept, θ₁ = slope.
Multiple Linear Regression
ŷ = θ₀ + θ₁x₁ + θ₂x₂ + … + θₚxₚ
p features, p+1 parameters.
OLS Slope (simple)
θ₁ = Σ(xᵢ − x̄)(yᵢ − ȳ) / Σ(xᵢ − x̄)²
Closed-form solution. Also: θ₁ = Cov(X,Y) / Var(X).
OLS Intercept (simple)
θ₀ = ȳ − θ₁x̄
The line always passes through (x̄, ȳ).
Normal Equation (matrix)
θ = (XᵀX)⁻¹ Xᵀy
Closed-form solution for multiple regression. O(n³) — slow for large p.
Cost Function (MSE)
J(θ) = (1/2m) × Σ(h(xⁱ) − yⁱ)²
The ½ is for cleaner derivatives. m = number of samples.
6 · Gradient Descent
Update Rule (general)
θⱼ := θⱼ − α × ∂J(θ)/∂θⱼ
α = learning rate. Repeat until convergence. Update ALL θ simultaneously.
Gradient for θ₀
∂J/∂θ₀ = (1/m) × Σ(h(xⁱ) − yⁱ)
Error × 1 (since x₀ = 1 always).
Gradient for θⱼ (j ≥ 1)
∂J/∂θⱼ = (1/m) × Σ(h(xⁱ) − yⁱ) × xⱼⁱ
Error × the feature value for that parameter.
Variants:
Batch GD: uses all m samples per update (slow but stable).
Stochastic GD: uses 1 random sample per update (fast but noisy).
Mini-batch GD: uses a batch of b samples (best of both worlds).
Feature Scaling (Min-Max)
x' = (x − min) / (max − min)
Scales to [0, 1]. Helps GD converge faster when features have different ranges.
Feature Scaling (Standardization)
x' = (x − μ) / σ
Mean 0, std 1. Preferred when data has outliers.
7 · Regression Metrics
MAE — Mean Absolute Error
MAE = (1/n) × Σ|yᵢ − ŷᵢ|
Same units as y. Robust to outliers. Not differentiable at 0.
MSE — Mean Squared Error
MSE = (1/n) × Σ(yᵢ − ŷᵢ)²
Units are y². Punishes large errors more. Differentiable.
RMSE — Root Mean Squared Error
RMSE = √MSE
Same units as y. Most commonly reported metric.
SStotal
SStotal = Σ(yᵢ − ȳ)²
Total variance in the data. Baseline: how bad is guessing the mean?
SSres (residual)
SSres = Σ(yᵢ − ŷᵢ)²
Variance your model can't explain. Smaller = better.
R² — Coefficient of Determination
R² = 1 − SSres / SStotal
% of variance explained. 1 = perfect, 0 = no better than mean, <0 = worse.
Adjusted R²
Adj R² = 1 − [(1−R²)(n−1) / (n−p−1)]
n = samples, p = features. Penalizes adding useless features.
MAPE
MAPE = (100/n) × Σ|yᵢ − ŷᵢ| / |yᵢ|
Percentage error. Undefined when yᵢ = 0.
8 · Logistic Regression
Sigmoid Function
σ(z) = 1 / (1 + e^(−z))
Maps any real number to (0, 1). Output = probability.
Hypothesis
h(x) = σ(θᵀx) = 1 / (1 + e^(−θᵀx))
P(y=1|x; θ). Classify as 1 if h(x) ≥ 0.5.
Log Loss (Binary Cross-Entropy)
J(θ) = −(1/m) × Σ[ yⁱ log(hⁱ) + (1−yⁱ) log(1−hⁱ) ]
Convex cost function for logistic regression. Heavily penalizes confident wrong predictions.
Decision Boundary
θᵀx = 0
Where σ(z) = 0.5. Linear boundary in feature space.
Odds
odds = p / (1 − p)
Ratio of success to failure probability.
Log-Odds (Logit)
log(p / (1−p)) = θ₀ + θ₁x₁ + … + θₚxₚ
Logistic regression is linear in log-odds space.
Softmax (multi-class)
P(y=k|x) = e^(zₖ) / Σⱼ e^(zⱼ)
Generalizes sigmoid to K classes. Outputs sum to 1.
9 · Classification Metrics
Accuracy
Accuracy = (TP + TN) / (TP + TN + FP + FN)
Misleading on imbalanced data. 95% accuracy means nothing if 95% of data is one class.
Precision
Precision = TP / (TP + FP)
"Of everything I called positive, how many actually were?" Minimize false alarms.
Recall (Sensitivity / TPR)
Recall = TP / (TP + FN)
"Of all actual positives, how many did I catch?" Minimize misses.
F1 Score
F1 = 2 × (Precision × Recall) / (Precision + Recall)
Harmonic mean. Balances precision and recall. Range: [0, 1].
Specificity (TNR)
Specificity = TN / (TN + FP)
"Of all actual negatives, how many did I correctly identify?"
False Positive Rate
FPR = FP / (FP + TN) = 1 − Specificity
X-axis of the ROC curve.
AUC-ROC
Area under the ROC curve (TPR vs FPR)
0.5 = random guess. 1.0 = perfect separation. Threshold-independent.
Log Loss (multi-class)
−(1/n) × ΣΣ yᵢₖ × log(pᵢₖ)
Lower is better. Penalizes confident wrong predictions harshly.
Confusion Matrix:
TP = true positive, TN = true negative, FP = false positive (Type I), FN = false negative (Type II).
All classification metrics are derived from these four numbers.
10 · Regularization
L1 — Lasso
J(θ) = MSE + λ × Σ|θⱼ|
Drives coefficients to exactly 0 → automatic feature selection. Sparse models.
L2 — Ridge
J(θ) = MSE + λ × Σθⱼ²
Shrinks coefficients toward 0 but never exactly 0. Handles multicollinearity.
Elastic Net
J(θ) = MSE + λ₁Σ|θⱼ| + λ₂Σθⱼ²
Combines L1 and L2. Best of both: sparsity + stability.
Dropout (neural networks)
During training: randomly set each neuron to 0 with probability p
Forces redundancy. Acts like an ensemble. Typically p = 0.2–0.5.
λ (lambda) = regularization strength.
λ = 0 → no regularization (back to plain OLS).
λ → ∞ → all coefficients → 0 (underfitting).
Find the sweet spot with cross-validation.
11 · Information Theory
Entropy
H(X) = −Σ P(xᵢ) × log₂P(xᵢ)
Measure of uncertainty/disorder. Max when all outcomes equally likely. Used in decision trees.
Gini Impurity
Gini = 1 − Σ pᵢ²
Alternative to entropy for decision trees. 0 = pure, 0.5 = max impurity (binary).
Information Gain
IG = H(parent) − Σ(nₖ/n) × H(childₖ)
Reduction in entropy from a split. Higher = better split.
KL Divergence
D_KL(P‖Q) = Σ P(x) × log(P(x)/Q(x))
How different Q is from P. Not symmetric: D(P‖Q) ≠ D(Q‖P).
Cross-Entropy
H(P,Q) = −Σ P(x) × log Q(x)
H(P,Q) = H(P) + D_KL(P‖Q). Used as loss function in classification.
Mutual Information
I(X;Y) = H(X) − H(X|Y)
How much knowing Y reduces uncertainty about X. 0 = independent.
12 · Linear Algebra Essentials
Dot Product
a · b = Σ aᵢbᵢ = |a||b|cos(θ)
Scalar result. Foundation of matrix multiplication and projections.
Matrix Multiplication
C = AB    where Cᵢⱼ = Σ Aᵢₖ Bₖⱼ
A is (m×n), B is (n×p) → C is (m×p). Not commutative: AB ≠ BA.
Transpose
(Aᵀ)ᵢⱼ = Aⱼᵢ
Swap rows and columns. (AB)ᵀ = BᵀAᵀ.
Inverse
A⁻¹A = AA⁻¹ = I
Only exists if det(A) ≠ 0 (square, full rank).
L1 Norm (Manhattan)
‖x‖₁ = Σ|xᵢ|
Sum of absolute values. Used in Lasso regularization.
L2 Norm (Euclidean)
‖x‖₂ = √(Σ xᵢ²)
Straight-line distance. Used in Ridge, KNN, and cosine similarity.
Cosine Similarity
cos(θ) = (a · b) / (‖a‖₂ × ‖b‖₂)
Measures direction similarity, ignores magnitude. Range: [−1, 1]. Common in NLP.
Eigendecomposition
Av = λv
v = eigenvector (direction preserved), λ = eigenvalue (scaling factor). Core of PCA.
13 · Bias–Variance Tradeoff
Total Error Decomposition
E[(y − ŷ)²] = Bias(ŷ)² + Var(ŷ) + σ²
σ² = irreducible noise. You can only reduce bias and variance — and reducing one tends to increase the other.
Bias
Bias(ŷ) = E[ŷ] − y_true
How far the average prediction is from the truth. High bias = underfitting (too simple).
Variance
Var(ŷ) = E[(ŷ − E[ŷ])²]
How much predictions change across different training sets. High variance = overfitting (too complex).
The goal: minimize total error, not just one component.
• Simple models → high bias, low variance (underfitting)
• Complex models → low bias, high variance (overfitting)
• Sweet spot → moderate bias, moderate variance (generalization)
K-Fold Cross-Validation
Split data into K folds. Train on K−1, test on 1. Rotate K times. Average scores.
Typical K = 5 or 10. Gives reliable estimate of generalization performance.
Train/Test Split
Typical: 80% train / 20% test
Or: 60% train / 20% val / 20% test
Never evaluate on training data. Never tune hyperparameters on test data.
