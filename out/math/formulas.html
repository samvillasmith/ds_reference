p-value
P(data ≥ observed | H₀ is true)P(datos ≥ observado | H₀ es verdadera)P(データ ≥ 観測値 | H₀ が真)
If p < α (usually 0.05), reject H₀.Si p < α (usualmente 0.05), rechazar H₀.p < α (通常 0.05) なら、H₀ を棄却。
Type I error (α): rejecting H₀ when it's actually true (false positive).
Type II error (β): failing to reject H₀ when it's actually false (false negative).
Power = 1 − β: probability of correctly rejecting a false H₀. Error Tipo I (α): rechazar H₀ cuando es verdadera (falso positivo).
Error Tipo II (β): no rechazar H₀ cuando es falsa (falso negativo).
Potencia = 1 − β: probabilidad de rechazar correctamente una H₀ falsa. 第一種の過誤 (α): H₀ が真なのに棄却 (偽陽性)。
第二種の過誤 (β): H₀ が偽なのに棄却失敗 (偽陰性)。
検出力 = 1 − β: 偽の H₀ を正しく棄却する確率。

Accuracy = (TP + TN) / (TP + TN + FP + FN)

Precision = TP / (TP + FP)
"Of everything I called positive, how many actually were?" Minimize false alarms."De todo lo que llamé positivo, ¿cuántos lo eran?" Minimizar falsas alarmas.「陽性と呼んだもののうち実際に陽性はいくつ？」誤報を最小化。

"Of all actual positives, how many did I catch?" Minimize misses."De todos los positivos reales, ¿cuántos atrapé?" Minimizar los fallados.「実際の陽性のうち、いくつ捉えたか？」見落としを最小化。

Harmonic mean. Balances precision and recall. Range: [0, 1].Media armónica. Balancea precisión y recall. Rango: [0, 1].調和平均。適合率と再現率のバランス。範囲: [0, 1]。

Todas las métricas de clasificación se derivan de estos cuatro números. 混同行列:

λ (lambda) = regularization strength.
λ = 0 → no regularization (back to plain OLS).

Find the sweet spot with cross-validation. λ (lambda) = fuerza de regularización.

Encuentra el punto óptimo con validación cruzada. λ (ラムダ) = 正則化の強さ。

13 · Bias–Variance Tradeoff 13 · Compromiso Sesgo–Varianza 13 · バイアス–分散トレードオフ
Total Error DecompositionDescomposición del Error Total総誤差の分解
E[(y − ŷ)²] = Bias(ŷ)² + Var(ŷ) + σ²
σ² = irreducible noise. You can only reduce bias and variance — and reducing one tends to increase the other.σ² = ruido irreducible. Solo puedes reducir sesgo y varianza — y reducir uno tiende a aumentar el otro.σ² = 削減不能なノイズ。バイアスと分散しか減らせず、一方を減らすと他方が増える傾向がある。
BiasSesgoバイアス
Bias(ŷ) = E[ŷ] − y_true
How far the average prediction is from the truth. High bias = underfitting (too simple).Qué tan lejos está la predicción promedio de la verdad. Alto sesgo = subajuste (demasiado simple).平均予測が真値からどれだけ離れているか。高バイアス = 過少適合 (単純すぎる)。
VarianceVarianza分散
Var(ŷ) = E[(ŷ − E[ŷ])²]
How much predictions change across different training sets. High variance = overfitting (too complex).Cuánto cambian las predicciones entre conjuntos de entrenamiento. Alta varianza = sobreajuste (demasiado complejo).訓練セットによって予測がどれだけ変わるか。高分散 = 過学習 (複雑すぎる)。
The goal: minimize total error, not just one component.
• Simple models → high bias, low variance (underfitting)
• Complex models → low bias, high variance (overfitting)
• Sweet spot → moderate bias, moderate variance (generalization) La meta: minimizar el error total, no solo un componente.
• Modelos simples → alto sesgo, baja varianza (subajuste)
• Modelos complejos → bajo sesgo, alta varianza (sobreajuste)
• Punto óptimo → sesgo moderado, varianza moderada (generalización) 目標: 一つの成分だけでなく総誤差を最小化。
• 単純なモデル → 高バイアス、低分散 (過少適合)
• 複雑なモデル → 低バイアス、高分散 (過学習)
• 最適点 → 中程度のバイアスと分散 (汎化)
