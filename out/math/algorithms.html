01 Linear Regression Regresión Lineal 線形回帰 Supervised · RegressionSupervisado · Regresión教師あり · 回帰

What it does. Fits the straight line (or hyperplane) that minimizes squared distance to the data points. The one equation every data scientist writes first: ŷ = β₀ + β₁x₁ + … + βₚxₚ. Qué hace. Ajusta la recta (o hiperplano) que minimiza la distancia al cuadrado a los puntos. La ecuación que todo científico de datos escribe primero: ŷ = β₀ + β₁x₁ + … + βₚxₚ. 何をするか。 データ点までの二乗距離を最小化する直線 (または超平面) を当てはめる。データサイエンティストが最初に書く式: ŷ = β₀ + β₁x₁ + … + βₚxₚ。

Use whenÚsalo cuando使う場面

You need a fast, interpretable baseline. Relationships look approximately linear. Stakeholders need to read the coefficients ("each extra year adds $2,400"). Necesitas una línea base rápida e interpretable. Las relaciones parecen aproximadamente lineales. Los interesados necesitan leer los coeficientes ("cada año extra suma $2,400"). 高速で解釈可能なベースラインが必要。関係がほぼ線形に見える。関係者が係数を読む必要がある (「年次が 1 年増えると $2,400 増加」)。

Avoid whenEvítalo cuando避ける場面

Relationships are nonlinear, features are heavily correlated (use Ridge), or you have many more features than observations (use Lasso or a regularized model). Las relaciones son no lineales, las variables están muy correlacionadas (usa Ridge), o tienes muchas más variables que observaciones (usa Lasso o un modelo regularizado). 関係が非線形、特徴量が強く相関している (Ridge を使う)、観測数より特徴量が多い (Lasso や正則化モデルを使う)。

02 Logistic Regression Regresión Logística ロジスティック回帰 Supervised · ClassificationSupervisado · Clasificación教師あり · 分類

What it does. Squashes a linear combination through the sigmoid function to output probabilities in (0, 1). Despite the name, this is a classifier — the "regression" is in log-odds space. Qué hace. Pasa una combinación lineal por la sigmoide para producir probabilidades en (0, 1). A pesar del nombre, es un clasificador — la "regresión" es en el espacio log-odds. 何をするか。 線形結合をシグモイドに通し、(0, 1) の確率を出力する。名前に反して分類器 — 「回帰」は対数オッズ空間での話。

Use whenÚsalo cuando使う場面

Binary or multi-class classification with interpretable coefficients. You need calibrated probabilities (not just labels). Baseline for any classification project. Clasificación binaria o multi-clase con coeficientes interpretables. Necesitas probabilidades calibradas (no solo etiquetas). Línea base para cualquier proyecto de clasificación. 解釈可能な係数が必要な二値・多クラス分類。ラベルだけでなく較正された確率が必要。分類プロジェクトのベースライン。

Avoid whenEvítalo cuando避ける場面

Classes aren't linearly separable in feature space and you don't want to engineer polynomial/interaction features. Use trees or a nonlinear kernel SVM instead. Las clases no son linealmente separables y no quieres ingeniar características polinómicas/interacciones. Usa árboles o SVM con kernel no lineal. 特徴空間でクラスが線形分離できず、多項式・交互作用特徴量を作りたくないとき。木や非線形カーネル SVM を使う。 

from sklearn.linear_model import LogisticRegression
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import make_pipeline
from sklearn.metrics import classification_report, roc_auc_score

model = make_pipeline(
    StandardScaler(),                        # sigmoid is scale-sensitive
    LogisticRegression(max_iter=1000, C=1.0)   # C = 1/λ, smaller = more regularization
)
model.fit(X_train, y_train)

probs = model.predict_proba(X_test)[:, 1]
preds = model.predict(X_test)
print(classification_report(y_test, preds))
print(f"AUC-ROC = {roc_auc_score(y_test, probs):.3f}")
03 K-Nearest Neighbors (KNN) K Vecinos Más Cercanos (KNN) k 近傍法 (KNN) Supervised · Lazy LearnerSupervisado · Perezoso教師あり · 怠惰学習

What it does. No training at all. To predict a new point, find the k training points closest to it and vote (classification) or average (regression). The entire dataset is the model. Qué hace. No hay entrenamiento. Para predecir un punto nuevo, encuentra los k puntos de entrenamiento más cercanos y vota (clasificación) o promedia (regresión). Todo el dataset es el modelo. 何をするか。 訓練をしない。新しい点を予測するには、最も近い k 個の訓練点を見つけて投票 (分類) または平均 (回帰)。データセット全体がモデルそのもの。

Use whenÚsalo cuando使う場面

Small-to-medium datasets with meaningful distance metrics. You want a nonlinear, nonparametric baseline. Recommendation systems ("users like you also bought…"). Datasets pequeños a medianos con métricas de distancia significativas. Quieres una línea base no lineal y no paramétrica. Sistemas de recomendación ("usuarios como tú también compraron…"). 意味のある距離指標がある中小規模データセット。非線形・ノンパラメトリックなベースラインが欲しい。推薦システム (「あなたに似たユーザーは…」)。

Avoid whenEvítalo cuando避ける場面

High-dimensional data (curse of dimensionality: distances become meaningless), huge datasets (prediction is O(n)), or latency-critical systems. Datos de alta dimensión (maldición de la dimensionalidad: las distancias pierden significado), datasets enormes (predicción O(n)), o sistemas con latencia crítica. 高次元データ (次元の呪い: 距離が意味を失う)、巨大データセット (予測が O(n))、レイテンシ重視のシステム。 

from sklearn.neighbors import KNeighborsClassifier
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import make_pipeline
from sklearn.model_selection import GridSearchCV

pipe = make_pipeline(
    StandardScaler(),                     # distance depends on scale — always scale!
    KNeighborsClassifier(weights="distance")
)

grid = GridSearchCV(pipe, {"kneighborsclassifier__n_neighbors": [3, 5, 7, 11, 21]}, cv=5)
grid.fit(X_train, y_train)

print(f"Best k = {grid.best_params_}")
print(f"Test accuracy = {grid.score(X_test, y_test):.3f}")
04 Naive Bayes Supervised · ClassificationSupervisado · Clasificación教師あり · 分類

What it does. Applies Bayes' theorem with a "naive" assumption: all features are independent given the class. That assumption is usually wrong — and yet it works remarkably well, especially for text. Training is a single pass: count frequencies. Qué hace. Aplica el teorema de Bayes con un supuesto "ingenuo": todas las variables son independientes dada la clase. El supuesto suele ser falso — y aun así funciona sorprendentemente bien, especialmente en texto. El entrenamiento es una pasada: contar frecuencias. 何をするか。 「素朴な」仮定 (クラスが与えられれば全特徴量は独立) のもとベイズの定理を適用。通常この仮定は成り立たないが、特にテキストでは驚くほどうまく動く。訓練は頻度を数える 1 パスのみ。

Use whenÚsalo cuando使う場面

Text classification (spam, sentiment, topics). Very large feature counts. You need blazing-fast training and inference, and can tolerate imperfect probability calibration. Clasificación de texto (spam, sentimiento, temas). Muchísimas características. Necesitas entrenamiento e inferencia ultrarrápidos y toleras una calibración imperfecta. テキスト分類 (スパム、感情、トピック)。特徴量が非常に多い。超高速の訓練・推論が必要で、確率の較正が完璧でなくてもよい。

Avoid whenEvítalo cuando避ける場面

Features are strongly dependent (Naive Bayes will double-count the evidence). Probability estimates need to be trusted for ranking or thresholding. Las variables son muy dependientes entre sí (Naive Bayes contará dos veces la evidencia). Las probabilidades deben ser confiables para rankear o fijar umbrales. 特徴量が強く依存している (証拠を二重カウントする)。ランキングや閾値設定で確率を信頼する必要があるとき。 

from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB

model = make_pipeline(

    MultinomialNB(alpha=0.1)                 # alpha = Laplace smoothing

model.fit(X_train_text, y_train)

05 Support Vector Machine (SVM) Máquina de Vectores de Soporte (SVM) サポートベクターマシン (SVM) Supervised · Classification/RegressionSupervisado · Clasificación/Regresión教師あり · 分類/回帰

What it does. Finds the maximum-margin hyperplane separating classes. Only the boundary points ("support vectors") matter. With the kernel trick, draws nonlinear boundaries by implicitly mapping to a higher-dimensional space — no need to compute that space. Qué hace. Encuentra el hiperplano de máximo margen que separa las clases. Solo importan los puntos frontera ("vectores de soporte"). Con el truco del kernel, dibuja fronteras no lineales mapeando implícitamente a un espacio superior — sin calcularlo. 何をするか。 クラスを分離する最大マージン超平面を見つける。境界上の点 (「サポートベクター」) だけが重要。カーネルトリックにより、高次元空間へ暗黙的に写像して非線形境界を引く — その空間を実際に計算する必要はない。

Use whenÚsalo cuando使う場面

Mid-size datasets with many features (text, genomics, bioinformatics). Clear margin between classes. You're willing to tune C and kernel parameters. Datasets medianos con muchas características (texto, genómica, bioinformática). Margen claro entre clases. Estás dispuesto a ajustar C y los parámetros del kernel. 多数の特徴量を持つ中規模データセット (テキスト、ゲノミクス、バイオインフォマティクス)。クラス間に明確なマージンがある。C とカーネルパラメータの調整を厭わない。

Avoid whenEvítalo cuando避ける場面

Massive datasets (training is O(n²) to O(n³)). Highly imbalanced classes without weighting. You need calibrated probabilities without extra work (Platt scaling is not free). Datasets masivos (entrenamiento O(n²)–O(n³)). Clases muy desbalanceadas sin ponderar. Necesitas probabilidades calibradas sin trabajo extra (Platt scaling no es gratis). 巨大データセット (訓練が O(n²)〜O(n³))。重み付けなしの強い不均衡クラス。追加作業なしに較正された確率が必要 (Platt スケーリングは無料ではない)。 

from sklearn.svm import SVC
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import make_pipeline

model = make_pipeline(
    StandardScaler(),
    SVC(kernel="rbf", C=1.0, gamma="scale", probability=True)
)
model.fit(X_train, y_train)

print(f"Test accuracy = {model.score(X_test, y_test):.3f}")
# kernel options: "linear" (no kernel trick), "rbf" (default nonlinear), "poly"
# Rule of thumb: start with linear. Upgrade to RBF only if linear underfits.
06 Decision Tree Árbol de Decisión 決定木 Supervised · Regression/ClassificationSupervisado · Regresión/Clasificación教師あり · 回帰/分類

What it does. Recursively splits the feature space along one feature at a time, picking the split that most reduces impurity (Gini or entropy for classification, variance for regression). You can print the tree and read the rules out loud. Qué hace. Divide recursivamente el espacio de características eligiendo, en cada paso, la división que más reduce la impureza (Gini o entropía para clasificación, varianza para regresión). Puedes imprimir el árbol y leer las reglas en voz alta. 何をするか。 特徴空間を 1 つの特徴量で再帰的に分割し、不純度 (分類では Gini やエントロピー、回帰では分散) を最も減らす分割を選ぶ。木を出力してルールをそのまま読める。

Use whenÚsalo cuando使う場面

Interpretability is critical (regulated industries, stakeholders who need to audit decisions). Features are on wildly different scales. Mix of numerical and categorical features with no preprocessing. La interpretabilidad es crítica (industrias reguladas, interesados que necesitan auditar decisiones). Características en escalas muy distintas. Mezcla de numéricas y categóricas sin preprocesamiento. 解釈性が重要 (規制産業、意思決定の監査が必要な関係者)。特徴量のスケールが大きく異なる。前処理なしで数値とカテゴリが混在する。

Avoid whenEvítalo cuando避ける場面

You need high accuracy — a single tree almost always overfits. Reach for Random Forest or Gradient Boosting instead. Decision trees alone are rarely a production choice. Necesitas alta precisión — un árbol único casi siempre sobreajusta. Usa Random Forest o Gradient Boosting. Los árboles solos rara vez son opción de producción. 高い精度が必要 — 単一の木はほぼ必ず過学習する。Random Forest や勾配ブースティングを使う。単独の決定木は本番ではほぼ使われない。 

from sklearn.tree import DecisionTreeClassifier, export_text

model = DecisionTreeClassifier(
    max_depth=5,                # cap depth to prevent overfitting
    min_samples_leaf=20,         # every leaf must have ≥ 20 samples
    criterion="gini",          # or "entropy"
    random_state=42
)
model.fit(X_train, y_train)

print(export_text(model, feature_names=feature_names))   # human-readable rules
print(f"Test accuracy = {model.score(X_test, y_test):.3f}")
07 Random Forest Supervised · EnsembleSupervisado · Ensemble教師あり · アンサンブル

What it does. Trains hundreds of deep decision trees, each on a bootstrap sample of rows with a random subset of features. At predict time, all trees vote (classification) or average (regression). Variance reduction without raising bias — the magic of bagging. Qué hace. Entrena cientos de árboles profundos, cada uno con una muestra bootstrap y un subconjunto aleatorio de variables. Al predecir, todos los árboles votan (clasificación) o promedian (regresión). Reducción de varianza sin aumentar el sesgo — la magia del bagging. 何をするか。 数百の深い決定木を、行のブートストラップ標本とランダムな特徴量部分集合で訓練する。予測時は全木が投票 (分類) または平均 (回帰) する。バイアスを上げずに分散を減らす — バギングの魔法。

Use whenÚsalo cuando使う場面

Strong default on tabular data. Mixed feature types. You have little time to tune and need something that works out of the box. Feature importance comes free. Opción por defecto sólida en datos tabulares. Tipos de variables mixtos. Poco tiempo para ajustar y necesitas algo que funcione sin configuración. La importancia de variables es gratis. 表形式データで強力なデフォルト。混在型特徴量。調整の時間が少なく、そのままで動くものが必要。特徴量重要度が無料で得られる。

Avoid whenEvítalo cuando避ける場面

Maximum predictive performance required (boosting usually wins). Memory is tight. Model size matters for deployment. Very large datasets where training cost dominates. Se requiere máxima performance predictiva (el boosting suele ganar). Memoria ajustada. El tamaño del modelo importa para el deployment. Datasets muy grandes donde el costo de entrenamiento manda. 最大の予測性能が必要 (ブースティングが勝つことが多い)。メモリがタイト。デプロイでモデルサイズが重要。訓練コストが支配する超大規模データセット。 

from sklearn.ensemble import RandomForestClassifier
import pandas as pd

model = RandomForestClassifier(
    n_estimators=500,        # more trees = more stable, slower
    max_depth=None,          # let trees grow fully
    min_samples_leaf=2,
    max_features="sqrt",       # √p features per split (classic default)
    n_jobs=-1,                # use all CPU cores
    random_state=42
)
model.fit(X_train, y_train)

print(f"Test accuracy = {model.score(X_test, y_test):.3f}")

# Feature importance — which variables matter?
importance = pd.Series(model.feature_importances_, index=feature_names).sort_values(ascending=False)
print(importance.head(10))
08 Gradient Boosting (XGBoost / LightGBM) Gradient Boosting (XGBoost / LightGBM) 勾配ブースティング (XGBoost / LightGBM) Supervised · EnsembleSupervisado · Ensemble教師あり · アンサンブル

What it does. Builds trees sequentially — each new tree fits the residual errors of the current ensemble. Small, shallow trees added one at a time with a learning rate. The state of the art for tabular data; wins most Kaggle competitions. Qué hace. Construye árboles en secuencia — cada nuevo árbol se ajusta a los errores residuales del ensemble actual. Árboles pequeños y poco profundos añadidos con una tasa de aprendizaje. Estado del arte en datos tabulares; gana la mayoría de competencias de Kaggle. 何をするか。 木を逐次的に構築 — 各新木は現在のアンサンブルの残差を当てはめる。小さく浅い木を学習率とともに 1 本ずつ追加。表形式データで最先端、Kaggle コンペの多くを制する。

Use whenÚsalo cuando使う場面

Maximum accuracy on structured/tabular data. You have time to tune learning rate, depth, and regularization. Features are a mix of numerical and categorical. Máxima precisión en datos estructurados/tabulares. Tiempo para ajustar tasa de aprendizaje, profundidad y regularización. Mezcla de variables numéricas y categóricas. 構造化・表形式データで最高精度。学習率・深さ・正則化を調整する時間がある。数値とカテゴリが混在した特徴量。

Avoid whenEvítalo cuando避ける場面

Images, audio, or raw text — use neural networks. Small datasets (<1000 rows) — it'll overfit without very heavy regularization. Extreme interpretability requirements. Imágenes, audio o texto crudo — usa redes neuronales. Datasets pequeños (<1000 filas) — sobreajusta sin regularización fuerte. Requisitos extremos de interpretabilidad. 画像、音声、生テキスト — ニューラルネットを使う。小さいデータセット (<1000 行) — 強い正則化なしでは過学習する。極めて高い解釈性が要求される場合。 

from xgboost import XGBClassifier
from sklearn.model_selection import train_test_split

X_tr, X_val, y_tr, y_val = train_test_split(X_train, y_train, test_size=0.15, random_state=0)

model = XGBClassifier(
    n_estimators=2000,
    learning_rate=0.05,          # small η + many trees → smoother convergence
    max_depth=6,
    subsample=0.8,               # row sampling — adds stochasticity
    colsample_bytree=0.8,        # column sampling — decorrelates trees
    reg_lambda=1.0,               # L2 on leaf weights
    eval_metric="logloss",
    early_stopping_rounds=50,     # stop when val loss hasn't improved in 50 rounds
    random_state=42
)
model.fit(X_tr, y_tr, eval_set=[(X_val, y_val)], verbose=False)

print(f"Best iteration = {model.best_iteration}")
print(f"Test accuracy  = {model.score(X_test, y_test):.3f}")
09 K-Means Unsupervised · ClusteringNo Supervisado · Agrupamiento教師なし · クラスタリング

What it does. Partitions points into k clusters by alternating two steps: (1) assign each point to its nearest centroid, (2) recompute centroids as the mean of assigned points. Repeat until assignments stop changing. Fast and simple — but you must choose k. Qué hace. Divide los puntos en k clusters alternando dos pasos: (1) asigna cada punto al centroide más cercano, (2) recalcula centroides como la media de los asignados. Repite hasta que las asignaciones no cambien. Rápido y simple — pero debes elegir k. 何をするか。 2 ステップを交互に: (1) 各点を最も近い重心に割り当て、(2) 割り当てられた点の平均として重心を更新。割り当てが変わらなくなるまで繰り返す。高速で単純 — ただし k の指定が必要。

Use whenÚsalo cuando使う場面

Customer segmentation, color quantization, document grouping. Clusters are roughly spherical and similar in size. You have a guess (or a business answer) for k. Segmentación de clientes, cuantización de color, agrupamiento de documentos. Clusters aproximadamente esféricos y de tamaño similar. Tienes una idea (o razón de negocio) para k. 顧客セグメンテーション、色量子化、文書グルーピング。クラスタがほぼ球状でサイズが似ている。k の当たり (またはビジネス上の答え) がある。

Avoid whenEvítalo cuando避ける場面

Clusters have irregular shapes, varying densities, or outliers (use DBSCAN). Features are not scaled — K-Means is distance-based and will dominate on large-scale features. Los clusters tienen formas irregulares, densidades variables o atípicos (usa DBSCAN). Variables sin escalar — K-Means es sensible a la escala. クラスタの形が不規則、密度が異なる、外れ値がある (DBSCAN を使う)。特徴量が未スケーリング — K-Means は距離ベースでスケール大の特徴量が支配する。 

from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import silhouette_score

X_scaled = StandardScaler().fit_transform(X)     # always scale before K-Means

# Pick k via the elbow / silhouette — don't just guess
for k in [2, 3, 4, 5, 6, 7, 8]:
    km = KMeans(n_clusters=k, n_init=10, random_state=42).fit(X_scaled)
    print(f"k={k}  inertia={km.inertia_:.1f}  silhouette={silhouette_score(X_scaled, km.labels_):.3f}")

# Final model with chosen k
model = KMeans(n_clusters=4, n_init=10, random_state=42).fit(X_scaled)
labels = model.labels_
10 DBSCAN Unsupervised · Density ClusteringNo Supervisado · Agrup. por Densidad教師なし · 密度クラスタリング

What it does. Groups points that are packed densely together, marks lone points as noise. You don't specify k — instead you set ε (neighborhood radius) and min_samples (density threshold). Discovers arbitrarily shaped clusters. Qué hace. Agrupa puntos densamente empacados, marca los aislados como ruido. No especificas k — fijas ε (radio de vecindad) y min_samples (umbral de densidad). Descubre clusters de formas arbitrarias. 何をするか。 密集した点群をクラスタに、孤立点をノイズとしてマーク。k を指定する代わりに ε (近傍半径) と min_samples (密度閾値) を設定。任意形状のクラスタを発見する。

Use whenÚsalo cuando使う場面

Clusters are non-convex (crescents, rings, blobs of varied shapes). You don't know k ahead of time. Outlier detection is a goal — DBSCAN labels them as -1. Clusters no convexos (crecientes, anillos, formas variadas). No sabes k de antemano. La detección de atípicos es un objetivo — DBSCAN los etiqueta como -1. 非凸クラスタ (三日月、リング、多様な形状)。k が事前に分からない。外れ値検出が目的 — DBSCAN は -1 としてラベル付けする。

Avoid whenEvítalo cuando避ける場面

Clusters have very different densities — one ε won't work for all. High-dimensional data where distances concentrate and density becomes meaningless. Los clusters tienen densidades muy distintas — un solo ε no servirá. Datos de alta dimensión donde las distancias se concentran y la densidad pierde significado. クラスタの密度が大きく異なる — 単一の ε では対応できない。距離が集中して密度が意味を失う高次元データ。 

from sklearn.cluster import DBSCAN
from sklearn.preprocessing import StandardScaler
import numpy as np

X_scaled = StandardScaler().fit_transform(X)

model = DBSCAN(eps=0.5, min_samples=10).fit(X_scaled)
labels = model.labels_

n_clusters = len(set(labels)) - (1 if -1 in labels else 0)
n_noise    = list(labels).count(-1)
print(f"Clusters found = {n_clusters}")
print(f"Noise points   = {n_noise} ({n_noise / len(labels):.1%})")

11 PCA Unsupervised · Dimensionality ReductionNo Supervisado · Reducción de Dimensión教師なし · 次元削減

What it does. Finds the orthogonal directions of maximum variance in your data (principal components), then projects onto the top k of them. You compress p features into k < p while keeping most of the variance. Qué hace. Encuentra las direcciones ortogonales de máxima varianza (componentes principales) y proyecta sobre las k primeras. Comprimes p variables en k < p conservando la mayor parte de la varianza. 何をするか。 データの最大分散方向 (主成分) を直交軸として見つけ、上位 k 個に射影する。分散の大部分を保ちながら p 個の特徴量を k < p に圧縮する。

Use whenÚsalo cuando使う場面

Too many features, strong multicollinearity. You want to visualize high-dimensional data in 2D/3D. Speeding up downstream models. Denoising (keep top components, drop the rest). Demasiadas variables, multicolinealidad fuerte. Quieres visualizar datos de alta dimensión en 2D/3D. Acelerar modelos posteriores. Eliminar ruido (conserva los componentes principales, descarta el resto). 特徴量が多すぎる、強い多重共線性。高次元データを 2D/3D で可視化したい。下流モデルを高速化したい。ノイズ除去 (上位成分を残して残りを捨てる)。

Avoid whenEvítalo cuando避ける場面

Features are categorical. Nonlinear structure matters (try t-SNE or UMAP for visualization, autoencoders for compression). Interpretability of original features is required. Variables categóricas. La estructura no lineal importa (usa t-SNE o UMAP para visualizar, autoencoders para comprimir). Se requiere interpretar las variables originales. カテゴリ特徴量。非線形構造が重要 (可視化には t-SNE や UMAP、圧縮にはオートエンコーダ)。元の特徴量の解釈が必要。 

from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler
import numpy as np

X_scaled = StandardScaler().fit_transform(X)    # PCA is scale-sensitive

pca = PCA(n_components=0.95)                  # keep enough components for 95% variance
X_reduced = pca.fit_transform(X_scaled)

print(f"Original  = {X.shape[1]} dims")
print(f"Reduced   = {X_reduced.shape[1]} dims")
print(f"Variance explained per component: {np.round(pca.explained_variance_ratio_, 3)}")
print(f"Cumulative: {np.round(np.cumsum(pca.explained_variance_ratio_), 3)}")
12 Neural Network (MLP) Red Neuronal (MLP) ニューラルネット (MLP) Supervised · Deep LearningSupervisado · Aprendizaje Profundo教師あり · 深層学習

What it does. Stacks linear transformations and nonlinear activations (ReLU, tanh, sigmoid) into layers. Trained via backpropagation — compute gradient of loss w.r.t. every weight, update by stochastic gradient descent. Universal approximator: enough hidden units can represent any continuous function. Qué hace. Apila transformaciones lineales y activaciones no lineales (ReLU, tanh, sigmoide) en capas. Entrenada por retropropagación — gradiente de la pérdida respecto a cada peso, actualizado por descenso de gradiente estocástico. Aproximador universal: con suficientes unidades, puede representar cualquier función continua. 何をするか。 線形変換と非線形活性化 (ReLU、tanh、シグモイド) を層として積み重ねる。誤差逆伝播で訓練 — 損失の各重みに対する勾配を計算し、確率的勾配降下で更新。普遍近似: 隠れユニットが十分にあればどんな連続関数も表現できる。

Use whenÚsalo cuando使う場面

Images (CNNs), sequences/text (Transformers, RNNs), audio, and any problem with lots of training data. Patterns are genuinely nonlinear and feature engineering is painful. Imágenes (CNN), secuencias/texto (Transformers, RNN), audio, y cualquier problema con muchos datos. Los patrones son genuinamente no lineales y la ingeniería de características es dolorosa. 画像 (CNN)、系列/テキスト (Transformer、RNN)、音声、訓練データが大量にある問題。パターンが本当に非線形で、特徴量エンジニアリングが苦行のとき。

Avoid whenEvítalo cuando避ける場面

Small tabular datasets — gradient boosting will beat you with 1% of the compute. You need interpretability. You can't afford GPU time or patience for hyperparameter tuning. Datasets tabulares pequeños — gradient boosting te gana con 1% del cómputo. Necesitas interpretabilidad. No puedes pagar GPU ni tener paciencia para ajustar hiperparámetros. 小規模な表形式データセット — 勾配ブースティングが計算量 1% で勝つ。解釈性が必要。GPU 時間やハイパーパラメータ調整の忍耐がない。

    MLPClassifier(

        alpha=1e-4,                   # L2 regularization

        early_stopping=True,           # stop when val score stops improving
        random_state=42
    )
)
model.fit(X_train, y_train)
