The problem with raw errors: El problema con los errores crudos: 生の誤差の問題点: they cancel out. se cancelan. 相殺してしまう。

−0.8 + 0.6 + 1.0 + −0.6 + −0.2 = 0.0

The negatives and positives cancel perfectly. Sum = 0. That doesn't mean the model is perfect — it means raw errors are useless as a metric. We need to get rid of the signs. Los negativos y positivos se cancelan perfectamente. Suma = 0. Eso no significa que el modelo sea perfecto — significa que los errores crudos son inútiles como métrica. Necesitamos deshacernos de los signos. 負と正が完全に相殺します。合計 = 0。これはモデルが完璧という意味ではなく、生の誤差は指標として役立たないという意味です。符号を取り除く必要があります。
Two ways to kill the sign: Dos formas de matar el signo: 符号を消す2つの方法:   ① Take absolute value → |error| → that's MAE ① Tomar valor absoluto → |error| → eso es MAE ① 絶対値を取る → |誤差| → それが MAE ② Square the error → error² → that's MSE ② Elevar al cuadrado → error² → eso es MSE ② 誤差を二乗する → 誤差² → それが MSE
MAE — Mean Absolute Error MAE — Error Absoluto Medio MAE — 平均絶対誤差
MAE = (1/n) × Σ|yᵢ − ŷᵢ|

Take each error, drop the sign (absolute value), average them. Done. Toma cada error, descarta el signo (valor absoluto), promedia. Listo. 各誤差を取り、符号を落とし (絶対値)、平均を取る。完了。
Step 1 — Take absolute value of each error: Paso 1 — Tomar el valor absoluto de cada error: ステップ 1 — 各誤差の絶対値を取る:   House 1Casa 1家 1: |−0.8| = 0.8 House 2Casa 2家 2: |+0.6| = 0.6 House 3Casa 3家 3: |+1.0| = 1.0 House 4Casa 4家 4: |−0.6| = 0.6 House 5Casa 5家 5: |−0.2| = 0.2
Step 2 — Average them: Paso 2 — Promediar: ステップ 2 — 平均を取る:   MAE = ( 0.8 + 0.6 + 1.0 + 0.6 + 0.2 ) / 5   = 3.2 / 5   = 0.64
MAE = 0.64 → "On average, predictions are off by $0.64k ($640)" MAE = 0.64 → "En promedio, las predicciones están erradas por $0.64k ($640)" MAE = 0.64 → 「予測は平均して $0.64k ($640) ずれている」
Why use MAE? ¿Por qué usar MAE? なぜ MAE を使う？   ✓ Easy to understand — it's in the same units as y ($k) ✓ Fácil de entender — está en las mismas unidades que y ($k) ✓ 理解しやすい — y と同じ単位 ($k) ✓ Treats all errors equally — a $1k miss counts the same everywhere ✓ Trata todos los errores por igual — un error de $1k cuenta igual en todos lados ✓ 全ての誤差を等しく扱う — $1k の誤差はどこでも同じ重み ✓ Robust to outliers — one huge error doesn't dominate ✓ Robusto a valores atípicos — un error enorme no domina ✓ 外れ値に強い — 一つの大きな誤差が支配しない ✗ Not differentiable at 0 — harder to use in gradient descent ✗ No diferenciable en 0 — más difícil de usar en descenso de gradiente ✗ 0 で微分不可能 — 勾配降下法で使いづらい
MSE — Mean Squared Error MSE — Error Cuadrático Medio MSE — 平均二乗誤差
MSE = (1/n) × Σ(yᵢ − ŷᵢ)²

Take each error, square it, average them. Squaring kills the sign AND punishes big errors more. Toma cada error, elévalo al cuadrado, promedia. Elevar al cuadrado mata el signo Y castiga más los errores grandes. 各誤差を二乗し、平均を取る。二乗は符号を消し、さらに大きな誤差をより重く罰する。
Step 1 — Square each error: Paso 1 — Elevar al cuadrado cada error: ステップ 1 — 各誤差を二乗する:   House 1Casa 1家 1: (−0.8)² = 0.64 House 2Casa 2家 2: (+0.6)² = 0.36 House 3Casa 3家 3: (+1.0)² = 1.00 House 4Casa 4家 4: (−0.6)² = 0.36 House 5Casa 5家 5: (−0.2)² = 0.04
Step 2 — Average them: Paso 2 — Promediar: ステップ 2 — 平均を取る:   MSE = ( 0.64 + 0.36 + 1.00 + 0.36 + 0.04 ) / 5   = 2.40 / 5   = 0.48
MSE = 0.48 → "Average squared error is 0.48" MSE = 0.48 → "Error cuadrático promedio es 0.48" MSE = 0.48 → 「平均二乗誤差は 0.48」
Notice the units problem: Fíjate en el problema de las unidades: 単位の問題に注目:

Your y is in $k. You squared the errors, so MSE is in $k².
What's "$0.48 thousand-dollars-squared"? It doesn't mean anything intuitive. Tu y está en $k. Elevaste los errores al cuadrado, entonces MSE está en $k².
¿Qué es "$0.48 miles-de-dólares-al-cuadrado"? No significa nada intuitivo. y は $k です。誤差を二乗したので、MSE は $k² です。
「$0.48 千ドル二乗」とは何か？直感的には何も意味しません。

Why use MSE? ¿Por qué usar MSE? なぜ MSE を使う？   ✓ Differentiable everywhere — perfect for gradient descent ✓ Diferenciable en todas partes — perfecto para descenso de gradiente ✓ どこでも微分可能 — 勾配降下法に最適 ✓ Punishes large errors more — a 2.0 error costs 4× more than a 1.0 error ✓ Castiga más los errores grandes — un error de 2.0 cuesta 4× más que uno de 1.0 ✓ 大きな誤差をより重く罰する — 2.0 の誤差は 1.0 の誤差の 4 倍のコスト ✓ This is what the cost function J(θ) is based on ✓ En esto se basa la función de costo J(θ) ✓ コスト関数 J(θ) はこれに基づく ✗ Units are squared — not directly interpretable ✗ Las unidades están al cuadrado — no directamente interpretable ✗ 単位が二乗 — 直接解釈できない ✗ Sensitive to outliers — one huge error can dominate ✗ Sensible a valores atípicos — un error enorme puede dominar ✗ 外れ値に敏感 — 一つの大きな誤差が支配する可能性
How squaring punishes big errors: Cómo el elevar al cuadrado castiga los errores grandes: 二乗が大きな誤差をどう罰するか:   Error of 0.2 → squared = 0.04  (tiny) Error de 0.2 → al cuadrado = 0.04  (minúsculo) 誤差 0.2 → 二乗 = 0.04  (極小) Error of 1.0 → squared = 1.00  (25× bigger!) Error de 1.0 → al cuadrado = 1.00  (¡25× mayor!) 誤差 1.0 → 二乗 = 1.00  (25倍大きい!) Error of 5.0 → squared = 25.0  (625× bigger!) Error de 5.0 → al cuadrado = 25.0  (¡625× mayor!) 誤差 5.0 → 二乗 = 25.0  (625倍大きい!)   The bigger the miss, the more MSE cares about it. Mientras mayor el fallo, más lo castiga MSE. 誤差が大きいほど、MSE はそれを重視する。 MAE would treat these proportionally: 0.2, 1.0, 5.0. MAE los trataría proporcionalmente: 0.2, 1.0, 5.0. MAE はこれらを比例的に扱う: 0.2, 1.0, 5.0。
RMSE — Root Mean Squared Error RMSE — Raíz del Error Cuadrático Medio RMSE — 二乗平均平方根誤差
RMSE = √MSE

That's it. Take MSE and square-root it. This undoes the squaring so units make sense again. Eso es todo. Toma MSE y sácale la raíz cuadrada. Esto deshace el cuadrado para que las unidades vuelvan a tener sentido. これだけ。MSE の平方根を取る。二乗を元に戻して、単位が再び意味を持つようにする。
Just one step — take the square root of MSE: Solo un paso — tomar la raíz cuadrada del MSE: 1 ステップだけ — MSE の平方根を取る:   RMSE = √MSE   = √0.48   = 0.6928
RMSE = 0.6928 → "Predictions are off by ~$0.69k ($693) on average" RMSE = 0.6928 → "Las predicciones están erradas por ~$0.69k ($693) en promedio" RMSE = 0.6928 → 「予測は平均して ~$0.69k ($693) ずれている」
Or expand the full formula: O desarrolla la fórmula completa: あるいは完全な式を展開:   RMSE = √[ (1/n) × Σ(yᵢ − ŷᵢ)² ]   = √[ (0.64 + 0.36 + 1.00 + 0.36 + 0.04) / 5 ]   = √[ 2.40 / 5 ]   = √0.48   = 0.6928
Why use RMSE? ¿Por qué usar RMSE? なぜ RMSE を使う？   ✓ Same units as y ($k) — directly interpretable ✓ Mismas unidades que y ($k) — directamente interpretable ✓ y と同じ単位 ($k) — 直接解釈可能 ✓ Still punishes big errors more (from the squaring inside) ✓ Aún castiga más los errores grandes (por el cuadrado interno) ✓ 内部の二乗により、大きな誤差を重く罰する ✓ Most commonly reported metric in ML papers ✓ La métrica más reportada en artículos de ML ✓ ML 論文で最も多く報告される指標 ✗ Still sensitive to outliers (like MSE) ✗ Sigue siendo sensible a valores atípicos (como MSE) ✗ 依然として外れ値に敏感 (MSE と同様)

Why is RMSE > MAE? ¿Por qué RMSE > MAE? なぜ RMSE > MAE？   RMSE is always ≥ MAE. The squaring step amplifies larger errors, RMSE siempre es ≥ MAE. El paso de elevar al cuadrado amplifica los errores mayores, RMSE は常に ≥ MAE。二乗ステップが大きな誤差を増幅し、 so the average gets pulled up. The bigger the gap between así que la media se eleva. Cuanto mayor la brecha entre 平均が引き上げられる。RMSE と MAE の差が RMSE and MAE, the more variance in your errors (some big, some small). RMSE y MAE, mayor es la varianza en tus errores (unos grandes, unos pequeños). 大きいほど、誤差の分散が大きい (大小が混在)。   If RMSE ≈ MAE → errors are all similar sizes Si RMSE ≈ MAE → los errores tienen tamaños similares RMSE ≈ MAE なら → 誤差が似たサイズ If RMSE >> MAE → a few big errors are dragging it up Si RMSE >> MAE → unos pocos errores grandes lo arrastran hacia arriba RMSE >> MAE なら → 少数の大きな誤差が引き上げている
